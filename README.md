# cubesat_image_classification
In the context of earth observation using cubesats, we noticed the squander of much energy due to unneeded pictures sent to earth.As some pictures are blurry, others are too rights and others are completely dark, thus, serve for nothing.
This project uses an ESP32-CAM to perform real-time cubesat image classification using Edge Impulse.
